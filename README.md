# idaproject-test

## DEV

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# JSON DB with hot reload at localhost:3030
$ npm run db

```

## PROD

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run build && npm run start

# JSON DB with hot reload at localhost:3030
$ npm run db

```
